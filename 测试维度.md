# 测试维度

本文档用于“辩论版 MBTI / 辩论人格测试”的后续开发，整理目前已经确认的 **8 个核心维度** 与 **16 个底层维度**，并统一命名、定义与合成逻辑，方便后续进行：

- 题目设计
- 计分建模
- 结果页文案撰写
- 前后端字段对接

---

## 一、整体结构

当前测试采用 **8 个核心维度 + 16 个底层维度** 的结构。

- **核心维度**：用于八边形雷达图展示，要求名字短、方向明确、一眼能懂。
- **底层维度**：用于实际出题与计分，每个核心维度由 2 个底层维度共同决定得分。

推荐规则：
- 每个核心维度先由对应 2 个底层维度简单平均得到
- 若后续测试发现某个底层维度解释力更强，可再做加权

---

## 二、8 个核心维度总览

| 序号 | 核心维度 | 高分方向 |
|---|---|---|
| 1 | 竞技导向 | 更把辩论当作比赛、博弈与赛果竞争 |
| 2 | 压迫倾向 | 更主动开战、追问、施压、压缩对手空间 |
| 3 | 现实锚点 | 更依赖现实世界锚点、事实材料与现实议题 |
| 4 | 评委导向 | 更围绕评委 / 裁判组织表达与理解比赛结果 |
| 5 | 辩棍浓度 | 辩论在生活中的参与频率与渗透度更高 |
| 6 | 放飞程度 | 更容易整活、风格外放、情绪上头 |
| 7 | 协同倾向 | 更从整队与配合视角理解比赛 |
| 8 | 渲染倾向 | 更依赖场景、感染、包装与语言张力说服 |

---

## 三、核心维度与底层维度明细

### 1. 竞技导向
**核心含义**  
衡量一个人整体上更把辩论当作竞技场，还是更把辩论当作表达、关系与体验场。

**高分画像**  
更偏比赛脑、赢面脑、赛场脑。

**低分画像**  
更偏表达、关系、体验与共同体感。

#### 底层维度 A：表达—竞技倾向
**定义**  
你打辩论更想获得表达快感，还是竞技快感。

- 低分端：更喜欢表达、观点输出、自我投射
- 高分端：更喜欢竞技、博弈、赛果与赢面

#### 底层维度 B：赛场—队伍倾向
**定义**  
你更迷恋比赛现场本身，还是更依附辩论队与队友共同体。

- 低分端：更喜欢辩论队、队友、社群氛围
- 高分端：更喜欢辩论赛、赛事现场、赛场机制

**合成逻辑**  
表达—竞技倾向 和 赛场—队伍倾向 任一偏低，整体都应该偏低

### 2. 压迫倾向
**核心含义**  
衡量一个人在比赛中是否天然倾向于主动开战、连续追问、逼迫对方交代。

**高分画像**  
喜欢先手、抢节奏、追问到底。

**低分画像**  
更偏回应、整理、防守与后手处理。

#### 底层维度 A：开战倾向
**定义**  
你是否习惯主动发起攻防、抢先开线、先把对方拉进你的节奏。

- 低分端：不急着主动起战场，更愿意等对方出招再回应
- 高分端：喜欢先发制人，主动抓定义、抓漏洞、抓不回应

#### 底层维度 B：逼问强度
**定义**  
你在进入攻防后，是否会持续追问、缩小承诺、逼对方明确交代。

- 低分端：点到为止，不一定连续追杀
- 高分端：喜欢一路追到对方必须明确表态

**合成逻辑**  
压迫倾向 = 开战倾向 + 逼问强度（建议先简单平均）

### 3. 现实锚点
**核心含义**  
衡量一个人更偏好现实世界的事实锚点，还是更偏好抽象推演、机理与设定。

**高分画像**  
现实派、政策题爱好者、材料派。

**低分画像**  
抽象派、脑洞题爱好者、推演派。

#### 底层维度 A：事实—机理倾向
**定义**  
你更喜欢用数据、案例、现实事实建立论证，还是更喜欢用机理、逻辑和推演打穿问题。

- 低分端：偏机理、逻辑、链条推演
- 高分端：偏数据、案例、现实事实、可验证锚点

#### 底层维度 B：现实—设定口味
**定义**  
你更喜欢政策辩、社会议题、现实热点，还是更喜欢脑洞题、哲学题、抽象设定。

- 低分端：偏脑洞题、哲学题、抽象设定
- 高分端：偏政策题、社会议题、现实热点

**合成逻辑**  
现实锚点 = 事实—机理倾向 + 现实—设定口味（建议先简单平均）

### 4. 评委导向
**核心含义**  
衡量一个人是否更围绕评委 / 裁判来组织表达，并将比赛结果理解为更受裁判影响。

**高分画像**  
裁判脑、票路脑、很看评委打比赛。

**低分画像**  
更坚持自我表达，更相信表现本身决定比赛。

#### 底层维度 A：自我—裁判取向
**定义**  
你更倾向讲自己想讲的东西，还是更倾向讲评委想听、最容易投票的话。

- 低分端：更讲自己想讲的
- 高分端：更讲评委想听的、最有投票价值的

#### 底层维度 B：裁判—表现归因
**定义**  
你更认为比赛结果主要受评委影响，还是主要受个人 / 团队表现影响。

- 低分端：更相信表现本身决定比赛
- 高分端：更认为评委变量会显著左右结果

**合成逻辑**  
评委导向 = 自我—裁判取向 + 裁判—表现归因（建议先简单平均）

### 5. 辩棍浓度
**核心含义**  
衡量辩论在一个人生活中的参与频率、出现密度和日常渗透程度。

**高分画像**  
辩论高频用户、辩论人格外溢明显。

**低分画像**  
辩论更像阶段性活动，不是持续生活状态。

#### 底层维度 A：赛事活跃
**定义**  
你参与辩论赛的频率和密度有多高，辩论赛是否是你生活中的高频事件。

- 低分端：很少打比赛，赛事参与低
- 高分端：经常打比赛，赛事参与密度高

#### 底层维度 B：日常论辩
**定义**  
你在生活中主动进行辩论式表达、讨论、反驳和立场输出的频率有多高。

- 低分端：日常较少进入辩论表达状态
- 高分端：平时也很容易“辩起来”

**合成逻辑**  
辩棍浓度 = 赛事活跃 + 日常论辩（建议先简单平均）

### 6. 放飞程度
**核心含义**  
衡量一个人在辩论中是否更容易整活、风格放飞、情绪上头，进入高能量与高戏剧性状态。

**高分画像**  
抽象、会整活、容易热起来、风格外放。

**低分画像**  
克制、稳定、正常、平直。

#### 底层维度 A：整活浓度
**定义**  
你是否倾向在辩论中玩梗、造效果、制造传播感和内容感，让比赛不只是“正常说话”。

- 低分端：更正经、克制、少娱乐化
- 高分端：更爱玩梗、造梗、制造传播素材

#### 底层维度 B：上头程度
**定义**  
你是否容易被赛场情绪、攻防节奏、对方挑衅和临场波动卷入，更快进入高情绪状态。

- 低分端：稳定、不容易被带跑
- 高分端：容易热起来、情绪起伏更强

**合成逻辑**  
放飞程度 = 整活浓度 + 上头程度（建议先简单平均）

### 7. 协同倾向
**核心含义**  
衡量一个人更倾向以整队与配合视角理解比赛，还是更偏单兵作战与局部输出。

**高分画像**  
整队脑、配合脑、团队发动机。

**低分画像**  
单兵脑、个人突破型、独立作战倾向更强。

#### 底层维度 A：整队建构
**定义**  
你是否习惯从整支队伍的视角构建比赛，思考分工、主线、优先级和最终收束，而不只关注自己这一段。

- 低分端：更关注自己负责的点位
- 高分端：会主动思考整队如何赢、如何分工与收束

#### 底层维度 B：单兵—配合倾向
**定义**  
你更擅长个人突破、独立作战，还是更擅长和队友接线、补位、同步并让表达服务整体。

- 低分端：更偏单兵突破
- 高分端：更偏与队友配合、补位、接线与同步

**合成逻辑**  
协同倾向 = 整队建构 + 单兵—配合倾向（建议先简单平均）

### 8. 渲染倾向
**核心含义**  
衡量一个人在表达和说服上，更依赖逻辑链条与分析，还是更依赖场景、感染、包装与语言张力。

**高分画像**  
场景派、包装派、感染派。

**低分画像**  
分析派、链条派、朴素表达派。

#### 底层维度 A：链条—场景倾向
**定义**  
你更习惯通过逻辑链条、机制分析和因果扣合说服，还是更习惯通过场景构造、画面感和代入感说服。

- 低分端：偏逻辑链条、机制分析、因果扣合
- 高分端：偏场景构造、画面感、代入感

#### 底层维度 B：朴素—包装倾向
**定义**  
你更倾向于直白克制地表达，还是更倾向于经营措辞、节奏、修辞和语言张力。

- 低分端：偏朴素、直给、克制
- 高分端：偏包装、修辞、节奏与语言设计

**合成逻辑**  
渲染倾向 = 链条—场景倾向 + 朴素—包装倾向（建议先简单平均）

## 四、字段命名建议（供开发使用）

### 核心维度字段
- competitive_orientation = 竞技导向
- pressure_orientation = 压迫倾向
- reality_anchor = 现实锚点
- judge_orientation = 评委导向
- debate_immersion = 辩棍浓度
- abstraction_orientation = 放飞程度
- collaboration_orientation = 协同倾向
- rendering_orientation = 渲染倾向

### 底层维度字段
- expression_vs_competition = 表达—竞技倾向
- stage_vs_team = 赛场—队伍倾向
- initiation_tendency = 开战倾向
- pressing_intensity = 逼问强度
- fact_vs_mechanism = 事实—机理倾向
- reality_vs_setting = 现实—设定口味
- self_vs_judge = 自我—裁判取向
- judge_vs_performance = 裁判—表现归因
- tournament_activity = 赛事活跃
- daily_argumentativeness = 日常论辩
- meme_intensity = 整活浓度
- emotional_heat = 上头程度
- team_construction = 整队建构
- solo_vs_coordination = 单兵—配合倾向
- chain_vs_scene = 链条—场景倾向
- plain_vs_stylized = 朴素—包装倾向
