DBTI 80题题库（初版）
一、统一选项
所有题统一使用 5 级量表：
-  A. 非常不符合我 
-  B. 比较不符合我 
-  C. 一半一半 / 看情况 
-  D. 比较符合我 
-  E. 非常符合我 

---
二、统一计分规则
正向题
题目越同意，越接近该维度的高分方向
-  A = 0 
-  B = 1 
-  C = 2 
-  D = 3 
-  E = 4 
反向题
题目越同意，越接近该维度的低分方向
-  A = 4 
-  B = 3 
-  C = 2 
-  D = 1 
-  E = 0

1. 表达—竞技倾向 expression_vs_competition
高分方向
更偏竞技、博弈、赛果与赢面
Q01
 打辩论时，赢一场通常比把自己真正想说的话完整说出来更让我满足。
 计分：expression_vs_competition 正向
Q02
 如果一场比赛输了，但我把自己真正认同的东西讲出来了，我通常也能接受。
 计分：expression_vs_competition 反向
Q03
 为了更稳地赢，我愿意放弃自己更喜欢、但不一定值票的论点。
 计分：expression_vs_competition 正向
Q04
 比起“最后有没有赢”，我更在意“我到底有没有讲成自己想讲的样子”。
 计分：expression_vs_competition 反向
Q05
 准备比赛时，我常常会先想“怎么更容易赢”，再想“我最想讲什么”。
 计分：expression_vs_competition 正向

三、开发建议
16.1 每个底层维度得分
每个底层维度取 5 题平均值。
例如：
- expression_vs_competition = (Q01 + Q02 + Q03 + Q04 + Q05) / 5 
- plain_vs_stylized = (Q76 + Q77 + Q78 + Q79 + Q80) / 5 

---
16.2 核心 8 维得分
再按之前定好的方式合成：
-  竞技导向 = expression_vs_competition + stage_vs_team
-  压迫倾向 = initiation_tendency + pressing_intensity
-  现实锚点 = fact_vs_mechanism + reality_vs_setting
-  裁判导向 = self_vs_judge + judge_vs_performance
-  辩论浸泡 = tournament_activity + daily_argumentativeness
-  抽象倾向 = meme_intensity + emotional_heat
-  协同倾向 = team_construction + solo_vs_coordination
-  渲染倾向 = chain_vs_scene + plain_vs_stylized
建议先用简单平均，后面内测再调权重。
